    # 学区房(划片)

    宝利花园 复兴区 9983.0元/㎡

    小区户型 楼间距超大，南侧商业楼，北侧住宅楼，住宅楼2梯5户，户型90平~130平，南北通透，方正

    小区设施 小区内部有500平方地上停车场，还有300平方花园，住户可以健身娱乐

    轨道交通 出小区南行200米是公交站家具大世界，有21路，71路，202路，60路等

    小区不足 小区规模小，只有一栋楼

    房屋品质 开发商是邯郸宝源房地产开发有限公司，交房时间是2013年，房屋质量非常好

    生活配套 小区南边2千米生活配套大型商超， 美食林，各大银行，清真肉联厂，公交站

    居民素质 小区居民素质高，租户很少，都是原住户

    小区户型 小区90平米和105平米三居室

    小区不足 小区不大，院内面积小，。

    房屋品质 开盘时间2010年9月份，2015年10月交房，

    附近学校 小区对口百花小学，十中中学

    生活配套 小区对门有王朗市场，紧邻人民路。

    青年苑小区 - 丛台

    小区户型: 主推户型，71平82平115平和136平，其次是132平113平78平，小区均为两梯5户，南北通透

    小区设施: 进院，地下车库，占地50亩，左手边前行20米，有自建花园，右手50米是，儿童游乐场，和健身器械

    房屋品质: 房子由邯郸市复兴房产开发有限公司开发，2007年拿地，2010年建设，2013年竣工2015年交房

    生活配套: 出门南行200米路南有美食林购物商场，在东行50米是邯郸第一附属医院，西行100米是邯郸市第一医院

    教育情况: 小学对口学校丛台小学，位置在本小区西行10米路东，初中是邯郸市25中西校区，现在是老25中初一

    小区不足: 小区规模太小，居住环境一般，道路拥挤

    龙盘锦绣花园 - 丛台 10575 元/m²

    小区户型 小区面积60-124平，，大部分是一提3户2居室，楼间距50米无遮挡采光充足。小区一共6栋。

    生活配套 出门向南20米是阳光超市，向东40米是工商银行和农业银行，向西60米是邯郸是中医院和三中学校。

    交通情况 小区距离公交站牌5米左右，有多条公交车如11路向西直达火车站，42路直达郝村，71路可直达北边新世纪

    教育情况 距离和平东小学20米，距离小区120米是邯郸市重点3中片内，环境优美，师资力量雄厚。

    小区不足 小区占地面积不大，绿化面积小。

    幸福馨苑 - 联纺路 10008 元/m²

    小区户型 小区均为2梯6户，主要户型90-52-68-80都是2室1厅，102为3室2厅1卫

    小区设施 小区西边为幼儿园里面有儿童游玩设施，空地800平

    生活配套 向北300米天天鲜超市，1000米路程是赵苑公园

    教育情况 对口五里铺小学，中学25中北校区

    小区不足 绿化面积不足，车位不足。

    丛台花园 - 丛台 15109 元/m²

    小区户型 丛台花园均为两梯四户，主推小户型89平适合刚需用户，119平，135平，三室两厅适合改善需求用户。

    小区设施 小区入口有一千平广场，广场旁有跷跷板，滑梯，单杠双杠等健身娱乐设施闲暇时光小朋友中老年人来此玩耍健身

    房屋品质 丛台花园于2014年动工2016年6月交房，纯钢混结构，采用国内前五的材料隔音防潮效果俱佳。东行50

    生活配套 东行一千米是邯郸市龙湖公园路东是邯郸市大剧院路南是环球中心即购物娱乐餐饮于一体。

    教育情况 距离邯郸市重点二十三中学200米，师资力量雄厚环境不错，距离黎明小学450米，市排名前五小学师资雄厚

    小区不足 目前地下车位还未完全开通，车位有些紧张，带物业开通后可以解决此问题。总价高，新证，税费高

    # 学区房(不划片)

    汉成华都 - 丛台

    小区户型 小区两梯五户居多，6/16/17号楼为两梯两户、两梯三户，主推83两居和93平小三居，适合刚需买房

    教育情况 小区内有幼儿园，世纪大街上有世纪路小学，东区东门口有真紫学校，升学制不划片需考试，教育资源充足

    小区不足 地上停车位不足，东西区中间容易堵车，物业费与物业服务质量不成正比，有待提升服务水平

    # 其他

    万浩吉祥 - 鑫港

    小区户型 商品房均为两梯四户，主力户型117三室两厅两卫，63平米72平米两居室 合适刚需客户

    小区设施 内部有绿化凉亭，儿童滑梯、跷跷板等娱乐设施，很多小朋友放学后在此玩耍交朋友。

    房屋品质 由邯郸市知名开发商建设，在邯郸有多个项目，2012年毛坯统一交房，分为商品房回迁房两部分。

    生活配套 门口有超市，菜市场，距离天鸿广场很近，可以满足日常购物需求

    交通情况 门口有20 24路公交车 直达火车站 可以满足日常出行需求。

    小区不足 小区商品房与回迁房在同一个院，人员比较混杂

    香兰雅居 - 丛台

    小区户型 香兰雅居有总共6栋楼、总高34层，面积59平米-179平米之间，楼间距50米，采光较好。

    生活配套 小区内部有超市，小型门诊，小区西行200米，邯郸市肿瘤医院

    小区不足 因为小区物业较低，小区内部卫生较差，所以小区物业不好。

    天泽园小区 - 鑫港 10585 元/m²

    小区户型 家和小区B区主要103㎡到133㎡，多层6层的房子较多 B区一共23栋楼, 楼间距40米 采光好.

    生活配套 小区北门往西500米有大型鑫港美食林 往西50米有大型农贸市场 往南800米设有邯郸市中心医院

    小区不足 小区位置偏向市区东北 没有对应小学

    创鑫华城 - 创鑫 11746 元/m²

    小区户型 小区共有八栋楼中间两户是93平米两室边户是140平米三室两厅两卫

    小区设施 小区有幼儿园幼儿园东北方向是一个活动场所有各种健身设施和座椅供小区内居民休息健身，近西门处有一个凉亭

    房屋品质 创鑫华城于2014交付使用，2018年办理房产证，小区是钢混结构，保温材料是国家一级标准。

    生活配套 小区南行50米邯郸银行总行，西行500米龙湖公园

    小区不足 小区2018年新办理的房产证过户费高
